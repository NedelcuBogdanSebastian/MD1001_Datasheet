1. Introduction

The MD-1001 data acquisition unit is designed to receive signals from sensors mounted on transformers or other monitored equipment. It processes the acquired/calculated parameters and transmits them to the end-user's server.

The MD-1001 (Fig. 1) is equipped with local data storage capabilities, maintaining a historical database of acquired parameters directly on the unit's internal solid-state drive.

Figure 1: MD-1001 Monitoring and Data Acquisition Unit

3. System Components

The MD-1001 system comprises a central acquisition module (Fig. 2) and up to four different types of plug-in signal conditioning modules. All secondary modules are powered directly by the central acquisition module.

Figure 2: Central Acquisition Module

The central module aggregates data from all connected sub-modules and hosts a MODBUS RTU server, making the data accessible via the RS-485 communication protocol.

3.1 Unified Signal Input Module

This module is available in three variants: 0-20 mA, 0-10 V, or 4-20 mA input.

Each module features 6 isolated channels. Additional modules can be installed to expand the system's capacity based on customer requirements.

  • For 0-20 mA & 4-20 mA modules: A precision 120 Ω resistor is used for current-to-voltage conversion on each channel.
  • For the 0-10 V module: A high-impedance 20 kΩ input is used for voltage measurement.
Figure 3: Unified Signal Input Module

3.2 PT100 RTD Input Module

The PT100 Module (Fig. 5) provides 4 channels for 3-wire PT100 RTD temperature sensors. The system can be expanded with additional modules for higher channel counts.

This module acquires temperature values and transmits them digitally to the central acquisition module.

Figure 5: PT100 RTD Input Module

3.3 Digital Input Module

The Digital Input Module provides 11 optically isolated digital input channels. These inputs are designed for dry contacts or voltage signals. Additional modules can be added for system expansion. Status data from this module is sent digitally to the central module.

Figure 6: Digital Input Module

3.4 Cooling Control Module

The Cooling Control Module (Fig. 4) features one 4-20 mA analog input (typically for Load Current on Phase B) and one PT100 RTD input (for Oil Temperature).

Using these inputs, the module calculates the winding temperature and automatically controls the cooling stages accordingly.

The module includes a real-time clock for stage rotation and can control up to 5 cooling stages via dry contact (relay) outputs. It transmits the acquired load current and oil temperature values back to the central module via a dedicated data interface.

Figure 4: Cooling Control Module

4.2 Wiring and Connection

Figure 8 illustrates the standard connection scheme for the MD-1001 system.

Figure 8: MD-1001 System Connection Overview

Connection Points:

  1. Primary Connections: Power Supply (5V DC), Ambient Temperature Sensor, MODBUS RS-485 Communication.
  2. Analog Inputs: Connection for 0-20 mA or 4-20 mA unified signal sensors.
  3. PT100 Inputs: Connections for 3-wire PT100 RTD temperature sensors.
  4. Digital Inputs: Connections for optically isolated digital signals.
WARNING: All electrical wiring and connection of the MD-1001 system must be performed exclusively by authorized Simtech International personnel.
